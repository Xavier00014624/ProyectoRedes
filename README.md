# Los Packet Tracer

## Proyecto de Redes Empresarial

Este repositorio contiene el desarrollo del proyecto grupal de redes del equipo **Los Packet Tracer**. El objetivo del proyecto es diseñar, justificar e implementar de forma simulada una red empresarial para una empresa ficticia distribuida en tres niveles.

El proyecto incluye el diseño físico y lógico de la red, subneteo VLSM por departamentos, presupuesto de implementación, selección de dispositivos, estrategia de enrutamiento y simulación funcional en GNS3 utilizando routers Cisco c3725.

## Objetivo general

Diseñar, presupuestar, justificar e implementar una solución de red empresarial funcional, escalable y ordenada para un edificio de tres niveles, aplicando conceptos de cableado estructurado, direccionamiento IP, VLSM, enrutamiento y simulación de redes.

## Descripción del proyecto

La empresa ficticia estará distribuida en tres niveles, cada uno con diferentes departamentos administrativos, técnicos y operativos. La red debe permitir la comunicación entre departamentos, mantener una estructura organizada y cumplir con los requisitos técnicos establecidos.

La solución contempla:

- Diseño físico de la red.
- Diseño lógico de la red.
- Distribución de dispositivos por departamento.
- Subneteo VLSM.
- Presupuesto de implementación.
- Justificación técnica de los insumos seleccionados.
- Simulación en GNS3.
- Configuración de routers Cisco c3725.
- Pruebas de conectividad.
- Estrategia de enrutamiento.
